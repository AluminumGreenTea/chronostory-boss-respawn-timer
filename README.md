# Chronostory · 怪物重生計時器

依「頻道」追蹤每一隻怪物 / BOSS 的重生時間。

## 功能

### 計時

- **新增 / 編輯 / 刪除**：選怪物圖片，或用「自訂圖示」搭配 10 種符號 × 8 種顏色（可加備註標籤）。
- **重生間隔**：小時 + 分鐘；選怪物時自動帶入該怪的**預設間隔**。
- **建立即倒數**：新增當下立即以現在時間開始倒數。
- **重置**：以「該怪預設間隔」重新起算（非手動改過的值）。
- **複製卡片**：頻道自動取「同種最大 +1」、間隔用預設值。
- **頻道就地編輯**：卡片上點擊輸入；聚焦後 `↑/↓ ±1`、`Shift+↑/↓ ±10`。
- **進度環**：右半圓漸層儀表 + 百分比，**以 5 分鐘為單位分色**、最後 1 分鐘轉金色，**倒數完成有漣漪**；超時 5 分鐘轉珊瑚紅。
- **自動排序**：可重生 → 倒數中（剩餘時間近到遠）。

### 管理 / 檢視

- **頻道總覽**（☰）：右側**推擠式抽屜**，依 BOSS 分組列出所有頻道，**重複標紅、可重生標色**，即時更新並保留捲動位置。
- **重複頻道檢查**：同一隻 BOSS 同頻道，新增 / 編輯時會擋下並提示。
- **搜尋**：名稱 / 備註 / 頻道（中英通用）。
- **篩選列**：出現 2 種以上怪物類型時，單行可左右捲動、依種類篩選。
- **鍵盤**：輸入欄 **Enter 直接建立**；ESC 可關閉目前視窗。

### 資料 / 其他

- **儲存**：localStorage；用結束時間換算，重整後倒數接得上。
- **是否記錄到此瀏覽器**：每筆可選（僅本次瀏覽 / 永久保存），記住上次選擇。
- **音效提醒** + 螢幕報讀（aria-live）。
- **主題**：深色、大地、星空（銀河 + 星點）、櫻緋、霧藍、抹茶、薰衣；下半圓弧滾輪切換。
- **雙語**：中 / 英切換。

## 架構

```
index.html            版面骨架，依序載入下列模組
css/style.css         樣式（含 7 主題、進度環、抽屜、動畫）
js/config.js          怪物清單、符號、顏色、LOGO、版號等靜態設定
js/theme.js           主題切換（滾輪選單）
js/i18n.js            中 / 英字典
js/storage.js         localStorage 存取
js/sound.js           WebAudio 提示音
js/timer.js           計時純函式（狀態計算、格式化、排序）
js/icons.js           內嵌 SVG 圖示
js/render.js          DOM 渲染（卡片、進度環、篩選、表單、頻道總覽）
js/app.js             主程式（狀態、事件、對齊整秒的 tick 迴圈）
```
