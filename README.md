# 楓之谷 · 怪物重生計時器

依「頻道」追蹤每一隻怪物 / BOSS 的重生時間。純前端、無需後端，
資料存在瀏覽器本機（localStorage），可直接部署到 GitLab Pages。

## 功能

- **新增計時器**：選怪物圖片，或用「自訂圖示」搭配 10 種符號 × 8 種顏色（可加備註標籤，不必打完整怪名）。
- **頻道**：輸入數字。一筆 = 一個頻道，同一隻怪要追多頻道就新增多筆。
- **重生間隔**：小時 + 分鐘（需大於 0）。
- **建立即倒數**：新增當下立刻以現在時間開始倒數，不用再按開始。
- **歸零轉狀態**：倒數到 0 自動變「可以重生了」，按鈕變「擊殺重置」，點一下以現在時間重新倒數。
- **卡片顯示**：圖示徽章 + 進度倒數環（藍色倒數中／金色可重生並脈動）、名稱／標籤、CH 頻道金標、間隔說明、倒數大字（可重生時顯示已超時多久）。
- **自動排序**：可重生 → 倒數中（剩餘時間近到遠）。
- **篩選列**：出現 2 種以上怪物類型時，可依種類只看該種所有頻道。
- **音效提醒**：重生時發出提示音，右上角鈴鐺可開關。
- **清除全部**：一鍵清空所有計時器。

## 架構（分離寫法）

```
index.html            版面骨架，依序載入下列模組
css/style.css         楓之谷風格樣式
js/config.js          怪物清單、符號、顏色等靜態設定
js/storage.js         localStorage 存取
js/sound.js           WebAudio 提示音
js/timer.js           計時純函式（狀態計算、格式化、排序）
js/render.js          DOM 渲染（卡片、進度環、篩選、表單）
js/app.js             主程式（狀態、事件、tick 迴圈）
assets/monsters/      怪物圖片（見資料夾內說明）
```

## 本機預覽

直接用瀏覽器開 `index.html` 即可（所有模組皆為傳統 script，
不需本機伺服器）。
